1) git init -> ініціалізація гіт репо
2) git status -> відобразити статус файлів в репо
3) git add filename -> додати файл в staging area
4) git commit -m "commit msg" -> зробити коміт
5) git log -> відобразити лог комітів/репо
6) git remote add origin git@github.com:YuliaSher/Java16M1L2.git
7) git branch -M main
8) git push -u origin main
9) git remote -v
